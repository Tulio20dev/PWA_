Principais novidades do Java 25

Java 25 é uma versão de suporte de longo prazo (LTS) — a Oracle fornecerá manutenção por vários anos

Várias melhorias de linguagem, API e da JVM para aumentar produtividade, desempenho, segurança e simplicidade.

O Java 25 foi oficialmente lançado em 16 de setembro de 2025 como uma versão LTS (Long-Term Support), o que significa que receberá atualizações de segurança e suporte por vários anos. Com essa nova versão, a plataforma Java traz melhorias significativas na linguagem, desempenho, segurança e produtividade para desenvolvedores de todos os níveis.

A nova funcionalidade permite escrever programas simples em Java de forma mais direta, sem a necessidade de declarar uma classe ou o tradicional public static void main(String[] args). Agora, é possível escrever apenas:

void main() { IO.println("Olá, Java 25!"); }
